# Product Admin Dashboard

Clean, modular admin dashboard.

## Structure
- index.html - Main entry
- js/app.js - App setup (< 100 lines)
- js/components/ - Managers (< 200 lines each)
- js/tabs/ - Tab modules (< 150 lines each)
- js/utils/ - Helpers (< 100 lines each)
- styles/ - CSS (< 100 lines each)

## Deploy
Upload to Cloudflare Pages or any static host.
